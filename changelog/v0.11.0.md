This release is the first release made from the new repo for the python package at [https://github.com/biojppm/rapidyaml-python](https://github.com/biojppm/rapidyaml-python). 

It is functionally equivalent to [c++/0.11.0](https://github.com/biojppm/rapidyaml/releases/tag/v0.11.0)), but with the following changes:

  - reduce the binary size of the source package (from 11.4MB to 0.5MB) by filtering unnecessary files
